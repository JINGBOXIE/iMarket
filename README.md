# iMarket Pro: AI-Powered Investment Engine

iMarket Pro is a sophisticated investment decision-support system designed to bridge the gap between raw financial data and actionable intelligence. Built with **Streamlit** and powered by **Gemini AI**, the platform provides a 360-degree view of US and Canadian equity markets.

## 🚀 Key Features
- **AI-Driven Analysis:** Real-time generation of risk assessment and opportunity reports.
- **Systematic Research:** Automated technical and fundamental analysis for 60+ global tickers.
- **User Governance:** Membership system with automated 24-hour query limits and RBAC.
- **Multi-Language Architecture:** Supports seamless switching between English and Chinese.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Intelligence:** Google Gemini AI API
- **Data Layer:** JSON Persistence (Local Experimentation Phase)
- **Language:** Python 3.13+
